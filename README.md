# 💳 HDFC Bank Finance Dashboard | Power BI

## 📌 Project Overview
The **HDFC Bank Finance Dashboard** is an interactive Power BI report designed to analyze banking transactions, customer performance, and financial trends. The dashboard provides valuable insights into revenue generation, transaction behavior, customer segmentation, and city-wise financial performance, helping stakeholders make data-driven decisions.

---

## 🚀 Dashboard Features

- 📈 Monthly Transaction Trend Analysis
- 💰 Revenue Contribution by Customer Segment
- 🔄 Transaction Count by Channel
- 🏙️ Top 5 Cities by Transaction Value
- 📊 Interactive KPIs and Cards
- 🎯 Dynamic Filters and Slicers
- 📉 Business Performance Visualization

---

## 📊 Key Insights

- Analyze monthly transaction growth and seasonal trends.
- Identify the most profitable customer segments.
- Compare transaction channels based on transaction count.
- Discover top-performing cities contributing the highest transaction value.
- Monitor important financial KPIs through interactive cards.
